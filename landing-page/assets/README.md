# assets — landing page

Brand assets owned by the landing page: `logo.svg`, `favicon.svg`. Copies of the
platform marks, kept here so the landing page is self-contained.

> Live pages reference the **served** copies under the platform static dir
> (`/static/logo.svg`, `/static/favicon.svg`) so there's one served source. These
> files are the editable design source — if you update a mark, update the copy in
> `../../static/` too (or copy it there at build/deploy).
